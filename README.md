# Ajax.page.loader.andRooter-jQuery.plugin

```html
<script src="src= http //ajax.googleapis.com/ajax/libs/jquery/1.11.3/jquery.min.js"></script>

<script src="path/to/ajaxPageLoaderAndRooter/lib.js"></script>

    });
</script>
```
